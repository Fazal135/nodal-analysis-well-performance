# Nodal Analysis: Well Performance Modeling

## 🔗 Live Demo

https://fazal135.github.io/nodal-analysis-well-performance/

## 📌 Overview

This project implements a complete **Nodal Analysis model** to evaluate hydrocarbon production performance.

It combines:

* Inflow Performance Relationship (IPR)
* Tubing Performance Relationship (TPR)

to determine the **natural operating point** of the well.

---

## ⚙️ Features

* Composite Vogel model for undersaturated reservoirs
* Multiphase TPR modeling using:

  * Standing’s correlation
  * Formation Volume Factors (Bo, Bg)
  * Blasius friction factor
* 50-step numerical integration along wellbore
* Interactive web-based simulator

---

## 📊 Results

* Operating Flow Rate: **249.3 sm³/day**
* Bottomhole Pressure: **208.8 bar**
